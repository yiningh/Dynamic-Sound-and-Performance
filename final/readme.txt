Final project for Dynamic Sound and Performance
==============================================

sound - 
Use MAX MSP. hit toggle to turn on the sound.

visualization - 
use openFrameworks v8.0. Src folder contains all the src codes, bin folder has the debug app.
required addons: ofxOpenCV, ofxOpticalFlowLK, ofxGui, ofxOsc. 

Use camera as an input for people to use body movements to manipulate sounds.
